# DigiTape Site

Static launch site for `digitape.co`.

## Free hosting options

- GitHub Pages: put these files in a public repo, enable Pages, and keep the `CNAME` file.
- Cloudflare Pages: connect the repo or upload this folder, then add `digitape.co` as a custom domain.

## Files

- `index.html`
- `styles.css`
- `CNAME`
- `assets/digitape-hero.png`
